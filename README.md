# Student Performance Analyzer :- A Data Analytics & Business Intelligence Project using Excel and Power BI

# Project Overview
The Student Performance Analyzer is a data analytics project designed to evaluate and visualize student academic performance using Excel and Power BI. The project focuses on identifying trends, performance gaps, and key factors influencing student outcomes to support data-driven academic decisions.

# Objectives
1. Analyze student performance data to uncover meaningful patterns and insights
2. Track key performance indicators (KPIs) such as grades, attendance, and pass rates
3. Identify high-performing and at-risk students
4. Support academic improvement strategies through visual reporting

# Tools & Technologies

1. Microsoft Excel (data cleaning, preprocessing, analysis)
2. Power BI (interactive dashboards, data visualization, KPI tracking)
